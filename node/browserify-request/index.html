this application demonstrates how to interact with ArcGIS Services without a clientside Esri SDK or a JavaScript framework like jQuery or Dojo.

click on the 'query' button below to return the value associated with an attribute of interest for an individual feature. update the text in the same field and click 'edit' to push a new value back to the server.
